# Registration-Form

- Projeto realizado através das aulas da Alura

## Tecnologias Utilizadas:

- ViteJS
- React JS
- Material UI
- JS

## Desenvolvimento:

- Layout Responsivo.
- Criação de um Form com validação no campo de CPF

## Como rodar a aplicação:

- Instale o [npm](https://nodejs.org/en/download/)
- Clone esse repositório
- Instale as dependências

```
npm install
```

- Rode a aplicação

```
npm run dev
```

- Acesse http://localhost:5173/
